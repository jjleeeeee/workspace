# [Usage] Menu

Menu는 사용자가 현재 맥락에서 선택할 수 있는 여러 액션이나 항목을 제공하기 위해 사용하는 오버레이 컴포넌트입니다.

## Context

Menu는 버튼이나 아이콘을 눌렀을 때 관련 선택지를 짧게 보여주는 데 사용해야 합니다. 독립적인 탐색 페이지나 긴 설정 목록처럼 많은 정보를 담아야 할 때는 별도 화면이나 Bottom Sheet를 사용해야 합니다.

## Principle

- Menu의 List는 List Item 컴포넌트를 사용해야 합니다.
- 클릭한 위치와 메뉴 노출 방향의 관계가 명확해야 합니다.
- 메뉴 높이가 길어질 경우 내부 스크롤로 처리해야 합니다.

## Usage Do / Don't

### List Item

Do

- Menu 안의 항목은 List Item 컴포넌트로 구성해야 합니다.

Don't

- 서로 다른 높이와 구조의 항목을 한 메뉴 안에 섞지 않아야 합니다. (스캔성 저하)

### Max Height

Do

- 메뉴가 max-height 370을 넘길 경우 메뉴 내부에서 스크롤해야 합니다.

Don't

- 메뉴 높이가 max-height 370을 초과하도록 노출하지 않아야 합니다. (화면 침범)

### Position

Do

- 사용자가 클릭한 영역을 기준으로 상단, 하단, 중앙 위치 규칙을 따라 노출해야 합니다.

Don't

- 트리거와 관계없는 위치에 Menu를 노출하지 않아야 합니다. (맥락 연결 약화)

### 선택지 범위

Do

- 현재 맥락에서 바로 선택 가능한 항목만 Menu에 넣어야 합니다.

Don't

- 여러 단계 설명이나 복잡한 설정 흐름을 Menu에 넣지 않아야 합니다. (정보 과밀)
