# [Usage] Chips

Chips는 옵션, 필터, 탭성 선택지를 작은 단위로 보여주고 사용자가 하나 또는 여러 개를 선택하기 위해 사용하는 선택 컴포넌트입니다.

## Context

Chips는 사용자가 빠르게 조건을 고르거나 콘텐츠 범주를 전환해야 할 때 사용해야 합니다. 목록 전체를 대체하는 컴포넌트가 아니라, 짧은 텍스트나 이미지가 포함된 선택 단위를 압축해 보여주는 용도로 사용해야 합니다.

## Principle

- 선택 가능한 옵션임을 알 수 있도록 상태와 크기를 일관되게 사용해야 합니다.
- 텍스트 타입과 이미지 포함 타입의 사이즈를 구분해야 합니다.
- 옵션이 많을 때도 사용자가 선택 상태를 잃지 않도록 배치해야 합니다.

## Usage Do / Don't

### Option Chips

Do

- Option Chips는 다수 옵션 중 단일 또는 다중 선택이 가능하도록 사용해야 합니다.

Don't

- 선택 불가능한 정보 라벨을 Option Chips처럼 사용하지 않아야 합니다. (조작 가능성 혼동)

### Wrapping

Do

- Option Chips가 페이지 너비를 초과하면 줄바꿈을 사용해야 합니다.

Don't

- 화면 밖으로 Chips가 잘리게 두지 않아야 합니다. (선택지 접근성 저하)

### Color

Do

- Chips의 텍스트와 아이콘 색상은 라이트/다크 모드에 맞게 반전되는 color 토큰을 사용해야 합니다.

Don't

- Chips의 텍스트와 아이콘 색상에 `fixed_color`를 사용하지 않아야 합니다. (다크 모드 색상 반전 불가)

### Tabs Chips

Do

- Tabs에서 사용되는 Chips의 사이즈는 Small로 사용해야 합니다.

Don't

- Text만 사용된 Chips에 Medium 사이즈를 사용하지 않아야 합니다. (시각 밀도 불균형)

### Image

Do

- 이미지가 포함된 타입은 Medium 사이즈를 사용해야 합니다.

Don't

- Chips 안의 이미지를 Avatar 속성처럼 사용하지 않아야 합니다. (컴포넌트 역할 혼동)
