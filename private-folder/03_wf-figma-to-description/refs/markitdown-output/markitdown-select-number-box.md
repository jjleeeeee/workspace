# [Usage] Select Number Box

Select Number Box는 사용자가 선택한 요소에 숫자나 순서를 표현하기 위해 사용하는 상태 표시 컴포넌트입니다.

## Context

Select Number Box는 이미지 편집 모드처럼 여러 요소를 선택하고 그 순서가 결과에 영향을 주는 화면에서 사용해야 합니다. 단순 선택 여부를 표시하는 용도가 아니라, 선택된 순번을 전달하는 용도로 사용해야 합니다. 숫자는 2자리수까지 표현해야 하며, 3자리수가 되는 경우에는 99+로 표현해야 합니다. 상태는 Selected, Selected-99+, Unselected로 구분해야 하며, 이미지 편집 모드에서는 배경 대비를 유지하기 위해 fixed_color를 사용해야 합니다.

## Principle

- 선택된 요소와 선택 순서를 동시에 이해할 수 있어야 합니다.
- 숫자는 선택 상태와 직접 연결되어야 합니다.
- 숫자는 2자리수까지 표현하고, 3자리수부터는 99+로 표현해야 합니다.
- 이미지 편집 모드에서는 fixed_color를 사용해야 합니다.

## Usage Do / Don't

### 숫자 표현

Do

- 선택 순서는 2자리수까지 표현하고, 3자리수가 되는 경우에는 99+로 표현해야 합니다.

Don't

- 3자리 이상의 숫자를 그대로 노출하지 않아야 합니다. (박스 가독성 저하)

### 선택 상태

Do

- Selected 상태에는 선택 순서를 노출하고, Selected-99+ 상태에는 99+를 노출해야 합니다.

Don't

- Unselected 상태에 숫자를 노출하지 않아야 합니다. (선택 여부 혼동)

### Color

Do

- 이미지 편집 모드에서는 fixed_color를 사용해야 합니다.

Don't

- 이미지 편집 모드에서 default 컬러를 사용하지 않아야 합니다. (배경 대비 부족)

### 위치

Do

- 숫자는 어떤 요소에 속하는지 알 수 있도록 선택된 요소 안이나 가까운 위치에 배치해야 합니다.

Don't

- 숫자를 요소와 떨어진 곳에 배치하지 않아야 합니다. (대상 연결성 약화)

### 용도

Do

- 이미지 편집처럼 선택 순서가 중요한 화면에서 사용해야 합니다.

Don't

- 단순 체크 상태 표시를 위해 Select Number Box를 사용하지 않아야 합니다. (컴포넌트 의미 오용)
