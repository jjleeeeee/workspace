# [Usage] Scroll Bar

Scroll Bar는 화면에 모두 표시되지 않는 콘텐츠의 현재 위치를 보여주고 다른 위치로 이동하기 위해 사용하는 스크롤 컨트롤입니다.

## Context

Scroll Bar는 콘텐츠 영역이 화면보다 길거나 넓어 사용자가 현재 위치와 남은 양을 파악해야 할 때 사용해야 합니다. 특히 PC 환경에서는 콘텐츠 영역과 스크롤 트랙 영역을 분리해 조작 가능성을 명확히 해야 합니다.

## Principle

- Scroll Bar는 콘텐츠 영역과 분리되어 노출되어야 합니다.
- PC에서는 콘텐츠 영역과 스크롤 트랙 영역이 구분되어야 합니다.
- 스크롤 위치를 방해 없이 보여주되 콘텐츠 가독성을 침범하지 않아야 합니다.

## Usage Do / Don't

### 영역 분리

Do

- Scroll Bar는 콘텐츠 영역과 분리해 노출해야 합니다.

Don't

- Scroll Bar를 콘텐츠 위에 겹쳐 정보 읽기를 방해하지 않아야 합니다. (가독성 저하)

### PC

Do

- PC에서는 콘텐츠 영역과 스크롤 트랙 영역을 명확히 구분해야 합니다.

Don't

- PC에서 스크롤 트랙을 숨겨 조작 위치를 알 수 없게 하지 않아야 합니다. (탐색성 저하)

### 위치 표시

Do

- 사용자가 현재 콘텐츠 위치와 이동 가능성을 이해할 수 있게 해야 합니다.

Don't

- 스크롤 가능한 콘텐츠인데 위치 표시를 전혀 제공하지 않아야 합니다. (콘텐츠 양 인지 어려움)

### 조작

Do

- Scroll Bar는 콘텐츠 이동을 보조하는 컨트롤로 사용해야 합니다.

Don't

- Scroll Bar를 장식 요소처럼 고정된 그래픽으로 사용하지 않아야 합니다. (조작 가능성 혼동)
