## 네이버 KNOCKIT AD 탭

**파일**: `videos/네이버.MP4` (30초, 1126×2436, 60fps)

### 흐름 요약

1. **홈 화면** — 바텀 네비(쇼핑/홈/투데이/클립/마이) 노출. 투데이 탭 탭.
2. **투데이 → 경제 탭** — 상단 탭바: 경제/뉴스/엔터/`KNOCKIT^AD`. KNOCKIT 탭 탭.
3. **로딩 상태** — 검정 배경 + 3dot 컬러 로딩 인디케이터 (즉시 재생 아님, 1~2초 대기).
4. **AD 탭 재생 중** — 상단 탭 선택됨. 영상 + 하단 CTA 카드 + 바텀 네비 구조.
5. **CTA 탭** — 인앱 쇼핑 페이지(Naver Shopping 내부) 진입. 별도 바텀 네비로 교체.

### UI 상세

| 항목 | 내용 |
|---|---|
| 탭 표기 | `KNOCKIT` 텍스트 + 우상단 `AD` 수퍼스크립트 |
| 영상 비율 | 가로형 (16:9에 가까움) — 전면이 아님 |
| 광고 메타 | 좌상단 `AD` 뱃지 + `▶ 00:27 WIFI에서 재생 중` |
| 뮤트 버튼 | 우상단 (🔊×) |
| 바텀 네비 | **유지** — CTA 카드 아래 바텀 네비 그대로 노출 |
| CTA 카드 | 흰색 카드: 브랜드 로고 + 썸네일 + 텍스트 + chevron `>` |
| CTA 랜딩 | Naver Shopping 인앱 페이지 (외부 브라우저 아님) |

## 무신사 AD 탭

**파일**: `videos/무신사.MP4` (20초, 1126×2436, 60fps)

### 흐름 요약

1. **카테고리 시트** — 앱 진입 시 카테고리 시트 오픈 상태.
2. **홈 → AD 탭 진입** — 상단 탭바: 콘텐츠/추천/`♥️^AD`(아이콘형)/랭킹/세일/발매...
3. **AD 탭 재생 중** — 영상 풀스크린. 탭바~화면 하단 엣지까지 점유. 바텀 네비 없음.
4. **CTA 카드 출현** — 영상 하단에 다크 반투명 카드 슬라이드업.
5. **추천 탭으로 이동** — 일반 피드 복귀. 바텀 네비 재노출(브랜드/스냅/홈/좋아요/마이).

### UI 상세

| 항목 | 내용 |
|---|---|
| 탭 표기 | 브랜드 아이콘(♥️) + `AD` 수퍼스크립트 (텍스트 탭명 없음) |
| 영상 비율 | 세로형 (9:16) — 화면 전체 점유 |
| 광고 메타 | 좌상단 `광고 00:33` (회색 뱃지) |
| 뮤트 버튼 | 우상단 (음소거 아이콘) |
| 바텀 네비 | **완전 사라짐** — 영상이 하단 엣지까지 점유 |
| CTA 카드 | 다크 반투명: 브랜드 로고 + 광고명 + 부제 + chevron `>` |
| CTA 랜딩 | 미확인 (영상 내 CTA 탭 미포함) |
