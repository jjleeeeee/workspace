## 무신사 제작가이드 추가 스펙 (PDF p.3~14)

| CTA 버튼 | 서브 문구 | 공백 포함 21자 이내 / 컬러 #000000 |

### 탭 진입점 (무신사)
- 텍스트 조합형: 한글 최대 7자, 영문 최대 9자
- 로고: 최대 264×144px (단, 안전영역 height 60 이내)
- 로고 컬러: 화이트/블랙 2종 필수 / 투명 배경 png
- 탭 진입 애니메이션 3가지 타입 선택 가능

### 안전 영역 (무신사 브랜드판)
- 영상 전체: 1080×1620px
- 워터마크 안전 위치(세이프): 800×1212px
- 확장 영역: 좌우 각 140px
- CTA 하단 버튼 영역: 279px

### 앱 스플래시 이미지
- 사이즈: 1432×1792px / 세이프영역: 694×1280px / JPG, 1MB 이하

### 앱 스플래시 동영상
- 사이즈: 1080×1352px / 세이프영역: 524×960px
- 총 1.5초 (모션 0.8초 + 포즈 0.7초) / MP4 H.264 / 사운드 제거 / 1MB 이하

### 로고 사용 규정
- 무신사 로고와 브랜드 로고 좌우/상하 병기 금지
- 무신사 로고 Black & White만 허용
- 브랜드 로고 비율별 분류: Wide(4:1 초과) / Long(2:1까지) / Short(1:1까지) / Square(정방형)

## 인사이트 3개

### 1. 바텀 네비 처리가 광고 몰입도를 가른다

| 앱 | 바텀 네비 처리 |
|---|---|
| ABLY | 별도 영역 유지 (영상 위에 겹치지 않음) |
| 네이버 KNOCKIT | 유지 — CTA 카드 아래 그대로 노출 |
| 무신사 AD | **완전 제거** — 영상이 하단 엣지까지 점유 |
| 카카오톡 | **완전 제거** — 전폭 CTA 버튼만 |

→ 위버스 9:16 전면 영상은 **무신사/카카오톡 방식(탭 진입 시 바텀 네비 숨김)** 이 적합. 리퀴드 글래스 네비가 콘텐츠에 오버랩 중이므로 숨김 OR 반투명 유지 중 선택 필요.

---

### 2. CTA는 "흰색 카드 vs 다크 오버레이" 두 패턴만 존재

| 패턴 | 앱 | 특징 |
|---|---|---|
| 흰색 카드 | ABLY, 네이버 | 영상과 명확히 구분. 카드가 영상을 가림 |
| 다크 반투명 오버레이 | 무신사 | 영상 위에 올라옴. 몰입 방해 적음. #000 85% |
| 전폭 버튼 | 카카오톡 | 단순 CTA, 브랜딩 없음 |

→ 9:16 전면 영상 + 아티스트 크리에이티브 보존 목적이면 **다크 반투명 오버레이** 유리. 무신사 실측: #000000 + 85% opacity, CTA 영역 하단 279px 고정.

---

### 3. 광고 소재 SAFE AREA는 플랫폼마다 다르나 "하단 잘림" 패턴은 동일

| 플랫폼 | 전체 사이즈 | SAFE AREA | 하단 여백 |
|---|---|---|---|
| 네이버 EAST (동영상) | 828×1472px | 628×640px | 하단 454px |
| 네이버 EAST (스크롤확장) | 828×1472px | 628×1072px | 하단 200px |
| 무신사 브랜드판 | 1080×1620px | 800×1212px | 하단 279px (CTA 버튼 영역) |

→ 위버스도 CTA 오버레이 + 바텀 네비 영역을 고려한 **하단 safe zone 명시가 광고 소재 스펙에 필수**. 예상: 하단 15~20% (약 300~380px / 1920px 기준).

## 위버스 적용안 2개

### A. 바텀 네비 — 쇼케이스 탭 진입 시 숨김 + 제스처로 복귀

무신사 방식. 탭 진입 → 바텀 네비 페이드아웃 → 영상 하단 엣지까지 점유. 화면 하단 스와이프업 or 영상 탭 → 네비 일시 노출.

- **장점**: 9:16 영상 풀스크린 몰입, 리퀴드 글래스 충돌 회피
- **단점**: 탭 이동 방법 유저 학습 필요. 첫 진입 시 힌트 UI 필요.
- **관련 이슈**: 오픈이슈 #3 (CTA + 팝업 플레이어 + 바텀 네비 충돌)

### B. CTA 카드 — 다크 반투명 오버레이 (영상 하단 고정)

무신사 방식. 영상 하단 ~18% 영역에 다크 반투명 카드 고정. #000000 85% + 아티스트 로고 + 광고명 + chevron.

- **장점**: 영상 크리에이티브 최대 보존. 흰색 카드 대비 컷아웃 없음.
- **단점**: 영상 내 하단 텍스트/자막 가릴 수 있음 → 광고 소재 스펙에 "하단 15% safe zone" 명시 필요.
- **광고 소재 스펙 추가 필요**: 위버스 기준 safe zone 계산 (무신사: 하단 279px / 1620px = 17.2%)
- **관련 이슈**: 오픈이슈 #3 CTA 위치, 광고 소재 스펙 업데이트 검토.
