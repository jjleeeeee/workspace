## 2. 배경 & 목적

### 2.1 프리미엄 광고 구좌의 필요성

위버스는 브랜딩 목적 광고 수요가 높은 매체임에도, 현행 구좌는 구조적 한계 보유.

- 주요 광고주 카테고리: 뷰티·엔터·패션·식음료·게임 → **IP/모델 중심 비주얼 소재를 대형 공간에 임팩트 있게 노출**이 캠페인 핵심
- 현행 직광고 구좌 대부분 소형 띠배너 → 브랜딩 소재(모델 전신, 제품, 캠페인 메시지) 온전히 전달 불가
- 대형 구좌는 홈팝업이 유일하나 **영상 소재 운영 불가**
- 스플래시: v3.14.0에서 대형화 진행했으나 지면 특성상 매우 짧은 시간 노출 + **랜딩(CTA) 불가**

### 2.2 세로형 영상 구좌의 부재

- 틱톡·인스타그램 릴스·유튜브 쇼츠 표준화 → 광고주 세로형 소재 제작 비중이 가로형 상회
- 틱톡과 유사 유저 특성 보유한 위버스에 세로형 소재 집행 희망 광고주 多, 집행 불가로 수주 실패 사례 빈번
- 현행 영상 광고 구좌: 프리롤 논스킵(인스트림 가로형, 15초) 1종뿐

### 2.3 글로벌홈 탭 선정 이유

두 수요(대형 구좌 + 세로형 영상)를 동시에 충족하는 구조.

| 기존 구좌 | 한계 | 쇼케이스 개선 |
|---|---|---|
| 홈팝업 | 영상 집행 불가 | 전면 세로형 영상 지원 |
| 스플래시 | 극단적으로 짧은 노출 시간, CTA 불가 | 유저가 탭 선택 → 충분한 노출 시간 + CTA |
| 띠배너 | 소형, 브랜딩 임팩트 부족 | 9:16 전면 독점 |

유사 서비스 검증: 네이버 쇼케이스(CPD 5천만), 무신사 브랜드판(3일 7천만), 에이블리(1일 1,500만), 카카오 프로필풀뷰(CPT 3,500만) — 동일 구조에서 프리미엄 단가 이미 검증.

## 4. 사용자 시나리오

### 정상 흐름 (Happy Path)

**Step 1 — 글로벌홈 진입: 탭 노출**
- 유저가 글로벌홈에 진입
- 국가 타깃팅 대상 → 탭 영역에 광고주명(+ AD 마크) 탭 표시
- 탭 위치: 스크롤 없는 첫 페이지 내 배치 희망 (→ **오픈 이슈 #1**)
- 1일 1사 단독 점유, SOV 100%

**Step 2 — 쇼케이스 탭 터치: 전면 영상 진입**
- 광고주명 탭 터치 → 탭 전용 화면으로 진입
- 진입과 동시에 9:16 전면 영상 **자동재생 (기본 Mute)**
- 아이콘 탭 시 음소거 해제 (언뮤트)
- 영상 하단: 광고주 지정 CTA 텍스트 타이틀 + 서브타이틀 + CTA 버튼

**Step 3 — CTA 버튼 클릭: 랜딩**
- CTA 버튼 클릭 → 광고주 설정 URL로 랜딩 (→ **오픈 이슈 #6** 인앱 vs 외부 브라우저)

### 엣지 케이스 (미팅 도출)

| 케이스 | 현재 방향 |
|---|---|
| GAM 광고 미수신 | 탭 노출하지 않음 (탭 활성화는 광고 수신 조건부) |
| 1차 노출 지연 (레이아웃 시프트) | → **오픈 이슈 #8** |

## 6. 비기능 요구사항

### 성능

- 1차 탭 노출: 글로벌홈 진입 후 레이아웃 시프트 없이 즉시 렌더링 (서버 API 기반으로 GAM 응답 속도와 독립)
- 영상 프리로드: 탭 클릭 시 로딩 대기 없이 즉시 재생

### 접근성 (WCAG 2.1 AA)

- 텍스트 색 대비 ≥4.5:1, UI 요소 ≥3:1
- VoiceOver (iOS) / TalkBack (Android) 레이블: "광고주명 광고 탭", "음소거 해제 버튼" 등
- Dynamic Type / Scalable Text 대응
- 영상 대체 수단 (자막/설명) — 광고 소재 가이드에 명시 필요
- 색 단독 정보 전달 금지

### 플랫폼 (Native iOS / Android 공통)

| 항목 | iOS HIG | Android M3 |
|---|---|---|
| 터치 영역 (CTA 버튼) | ≥44pt | ≥48dp |
| Safe Area | 상하 Safe Area 확보 필수 | Edge-to-edge + inset |
| 제스처 | Swipe-back 시 광고 탭 탈출 정의 필요 | Back gesture / button |
| 오디오 세션 | AVAudioSession 카테고리 정의 | AudioFocus 정책 |
| 전면 영상 | AVPlayer / AVPlayerLayer | ExoPlayer |

## 9. 운영 정책

### 광고주 심사

- 카테고리 사전 심사: 브랜드 세이프티 및 팬덤 정서 충돌 방지 (경쟁 IP, 부적절 카테고리 차단)
- 선예약 우선 원칙 (취소 마감일, 위약 상세 기준 추후 확정)
- 소재 검수: 노출 N영업일 전 입고 및 검수 SLA 운영
- 대상 국가: 한국/일본 분리 판매, 단가 차등

### 노출 운영

- 직광고 판매 건 있을 경우에만 탭 노출 (미판매일 탭 미노출)
- GAM 스케줄링 기반 캠페인 종료 시 자동 비노출
- 1일 1사 SOV 100%, 동일일 복수 광고주 불가
- 탭 운영 기간 종료 후 별도 랜딩 리다이렉트 불필요 (광고 특성상 캠페인 종료 = 탭 소멸)

### 지표 공유

- 캠페인 종료 후 1주일 이내 Clog + 위버스 인사이트 지표 공유 (기간 한정탭 정책 준수)
