# 쇼케이스 광고 재생 화면 — 안 B (바텀 네비 유지)

> **핵심 가설**: 바텀 네비 유지 + FAB 위치 재배치 → 기존 UX 패턴 유지, 영상 가시 영역 축소 감수

> **개발 리스크 높음** — FAB 위치 동적 제어 개발팀 확인 필요

## 목표 & 사용자 작업

- 쇼케이스 탭 진입 후 광고 영상 재생 (풀블리드 아님 — 바텀 네비 영역 제외)

## 콘텐츠 & 컴포넌트

### 실제 영상 가시 영역 계산

```
전체 화면: 852pt (iPhone 15 Pro 기준)
상단 safe area + 메타: ~100pt
바텀 네비: ~83pt
CTA 오버레이: ~120pt
────────────────────
영상 가시 영역: 852 - 100 - 83 - 120 = 549pt / 852pt = 약 64%
```

→ 안 A(~82%) 대비 약 18%p 가시 영역 손실

### CTA 오버레이 내부 구조 (바텀 네비 위, 120pt 영역)

```
┌─────────────────────────────────────────┐  ← 바텀 네비 상단 경계
│  [Avatar 32pt]  브랜드명 15pt SemiBold  │
│                 카피 13pt Regular       │
│                                    [>]  │
└─────────────────────────────────────────┘  ← 바텀 네비 (별도 컴포넌트)
```

| 요소 | chord 컴포넌트 | 스펙 |
|---|---|---|
| 브랜드 로고 | Avatar / Circle / Size-S (32pt) | 크기 축소 (공간 제약) |
| 브랜드명 | 텍스트 직접 배치 | 15pt SemiBold, #FFFFFF |
| 카피 | 텍스트 직접 배치 | 13pt Regular, #FFFFFF, 1줄 |
| Chevron | Button / Icon Button / Ghost / Fixed / XSmall | icon=chevron-right |
| 뮤트 버튼 | Button / Icon Button / Ghost / Fixed / Medium | icon=volume-off |
| FAB (재배치) | 위치 변경 — 현재 위치 미정 (시안에서 옵션 2개 표시) | |

### FAB 재배치 옵션 (시안 내 A/B 분기)

- **옵션 B-1**: 영상 우하단 (CTA 오버레이 좌상단) — 영상과 CTA 경계에 위치
- **옵션 B-2**: 영상 영역 우측 중앙 고정 — CTA와 완전 분리

## UX 체크리스트

- [ ] 정보위계: 브랜드 로고 → 브랜드명 → 카피 → chevron
- [ ] 터치 영역: CTA 카드 ≥44pt / ≥48dp
- [ ] 바텀 네비 탭 이동 — 기존 패턴 유지 (학습 불필요)
- [ ] FAB 재배치 후 사용자 인지 가능 여부 — **테스트 필수**
- [ ] 영상 가시 영역 64% 수용 가능 여부 — 광고주 협의 필요
- [ ] 뮤트 상태 기본값: 뮤트 ON

- [ ] 개발: FAB 특정 탭 한정 위치 제어 — 네이티브 커스텀 레이아웃 필요

## Native 패턴 결정

- **iOS HIG**
  - 바텀 네비(`UITabBar`) 유지 — home indicator 위 83pt
  - CTA 오버레이는 TabBar 상단에 additional view로 stack
  - FAB 위치 동적 변경: layout constraint 업데이트 또는 부모 뷰 교체

- **Android M3**
  - NavigationBar 유지 (M3 NavigationBar 컴포넌트)
  - CTA overlay: NavigationBar 위 CoordinatorLayout 삽입
  - FAB 재배치: `layout_anchor` 속성 변경

## 개발 리스크 명세

| 리스크 | 영향 | 확인 필요 |
|---|---|---|
| FAB 위치 특정 탭 한정 제어 | 높음 | iOS LayoutConstraint / Android anchor 동적 변경 가능 여부 |
| 미니 배너 자동 노출 억제 | 중간 | 양 안 공통. 진입 탭 기준 억제 로직 존재 여부 |
| CTA + 바텀 네비 레이어 충돌 | 낮음 | Z-order 명확히 정의 시 해결 가능 |
