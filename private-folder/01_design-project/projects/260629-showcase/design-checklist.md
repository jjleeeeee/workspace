# 위버스 쇼케이스 — 디자이너 체크리스트

---

## 1. 글로벌홈 탭 영역

- [ ] **탭 위치 확정** — PM/서비스기획팀 합의 필요. 현재 `My Feed / Discover / Party / 쇼케이스 / Spot` 순 제안
- [ ] **탭 라벨 형태** — 텍스트(영문 10자 이내) vs 광고주 로고 이미지 중 결정. 이미지 시 fallback 텍스트 정책
- [ ] **AD 마크 디자인** — 위치, 사이즈, 컬러 (기존 AD 마크 스타일 Chord에서 확인)
- [ ] **탭 로딩 상태** — GAM 응답 전 탭 skeleton or placeholder 처리 방식
- [ ] **탭 강조 이펙트** — "주목도 향상 추가 이펙트" 구현 가능 여부 개발 확인 후 필요 시 디자인

---

## 2. 쇼케이스 탭 전면 화면 레이아웃

- [ ] **Safe Area 가이드 정의** — iOS / Android 공통 광고 소재 제작용 Safe Area 크롭 영역 명시 (평균 상하 20% 크롭 기준)
- [ ] **Safe Area 광고주 가이드 문서** — 소재 제작 시 주요 메시지/로고 배치 금지 영역 시각화
- [ ] **스틸컷 이미지 노출 구간** — 영상 로딩 중 스틸컷 표시 → 재생 시작 시 전환 방식 (fade or cut)
- [ ] **상단 Top Navigation + 탭 영역 유지** — 쇼케이스 탭 내에서 고정 노출. 레이아웃에서 z-index / 오버레이 처리 확인
- [ ] **하단 홈 인디케이터 (iOS)** — 광고 영상이 홈 인디케이터와 겹치지 않도록 safe area inset 처리

---

## 3. 영상 컨트롤 UI

- [ ] **음소거/해제 아이콘** — 위치, 사이즈, 배경 처리 (반투명 pill? 아이콘만?)
- [ ] **일시정지 아이콘** — 화면 탭 시 나타나는 위치와 auto-hide 타이밍
- [ ] **리플레이 버튼** — 영상 완료(100%) 시 노출 형태. 영상 위 오버레이? 중앙 배치?
- [ ] **재생 진행 표시** — progress bar 여부 (스펙 미정, 결정 필요)

---

## 4. CTA 버튼 영역

- [ ] **CTA 레이아웃** — 아이콘(`%appIcon%`) + 타이틀(15자) + 서브타이틀(20자) 배치
- [ ] **CTA 하단 고정 여부** — 영상 하단 고정 vs 스크롤 불가 전면 화면이므로 항상 노출
- [ ] **텍스트 말줄임 처리** — 15/20자 초과 or 광고주 텍스트 길이 가변 케이스 처리
- [ ] **다국어 대응** — 한글/일어 텍스트 길이 차이 시 레이아웃 깨짐 방지

---

## 5. 팝업플레이어 중첩 상태

- [ ] **팝업플레이어 기본형/축약형 오버레이** — CTA 버튼과 중첩되어도 허용 (정책 확정). 드래그로 이동 가능하므로 별도 처리 없음 → 디자인 문서에 명시
- [ ] **리스닝파티 팝업** — 동시 사운드 허용 예외. 디자인 관점 처리 없음. 개발 확인 후 스펙 반영

---

## 6. 빈 상태 / 에러 / Fallback

- [ ] **쇼케이스 탭 미노출 케이스** — 탭 자체가 사라지므로 별도 empty state UI 불필요. 문서에 명시
- [ ] **스틸컷 이미지 미수신 케이스** — 영상 로딩 전 fallback 처리 정의

---

## 7. 광고 소재 스펙 가이드 (광고주용)

- [ ] **소재 규격 문서** — 1080×1920, ≤30MB, 6~15초 권장 / 최대 30초
- [ ] **Safe Area 가이드 이미지** — 상하단 크롭 기준 + 주요 배치 금지 영역 시각화
- [ ] **로고 이미지 규격** — 탭 내 노출용 (`%logo%`) 사이즈, 포맷, 배경 투명 여부
