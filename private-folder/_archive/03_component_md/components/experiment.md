## Component

Radio 컴포넌트는 두 가지 표시 환경에서 기본, 선택, 비활성 상태를 보여줍니다.

| 구분 | Default | Enabled | Disabled |
| --- | --- | --- | --- |
| Light/Dark | 선택되지 않은 기본 상태<br>`59215:200966` | 선택된 상태<br>`59215:200968` | 비활성 상태<br>`59215:200970` |
| Fixed | 어두운 배경 위의 기본 상태<br>`59215:200972` | 어두운 배경 위의 선택 상태<br>`59215:200974` | 어두운 배경 위의 비활성 상태<br>`59215:200976` |

## 1. Guide

### Padding System : Radio + Text

예시 영역 안에 Radio와 Text를 가로로 배치합니다.

- 예시 영역은 연한 회색 박스입니다.
- Radio와 Text는 예시 영역 중앙에 배치합니다.
- Radio는 선택된 상태 variant `59215:200968`을 사용합니다.
- Text는 `텍스트를 입력해주세요`를 사용합니다.
- Radio와 Text 사이 간격은 `padding/box/75`로 표시합니다.
- 간격 표시선은 Radio와 Text 사이에 배치합니다.
