# Icons

> Chord Design System 아이콘 카탈로그. 네이밍 규칙: `ic_{name}`. 24px 기준으로 사용하며, 필요시 스케일 조정.

---

## 크기 체계

아이콘은 **24px(medium) 단일 기준**으로 사용하며 레이아웃에 따라 스케일로 조절한다.

| 크기 | px | 주요 사용처 |
|------|----|------------|
| tiny | 10px | 극소형 배지 |
| xxsmall | 12px | 배지, 태그 내부 |
| small | 20px | 버튼 내부, 보조 |
| medium | 24px | **기본 크기** — 탭바, 버튼, 일반 UI |
| large | 32px | 강조 아이콘 |
| xlarge | 40px | 대형 버튼 |
| 2xlarge | 48px | 섹션 헤더 |
| 3xlarge | 56px | 히어로 |
| 4xlarge | 64px | 랜딩, 온보딩 |

## 버튼 슬롯 사용 가이드

Button의 Option-Leading / Trailing-Icon 슬롯에 사용할 아이콘은
이 카탈로그에서 선택한다. 아이콘은 24px을 기준으로 사용하며 버튼 크기에 따라 스케일 조정.

| Button Size | 아이콘 참조 크기 |
|-------------|----------------|
| XLarge(52) / Large(44) / Medium(40) | 24px (medium) |
| Small(36) / XSmall(32) | 24px → scale 0.83 |
| XXSmall(24) | 24px → scale 0.67 |
