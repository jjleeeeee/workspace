# 캐시
> 보유 캐시를 조회하고 사용 금액을 입력하는 섹션

## 구성

- 섹션 헤더: `"캐시"` `body-lg/system-700` (접기 아이콘 없음)
- 보유 캐시 잔액: `body-s/system-400`, color=`system/color/text/secondary`
- `cashBalance=0`이면 Text Input 비활성
