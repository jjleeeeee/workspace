## Visual Takeaways

1. Use fewer large containers and more disciplined small repeated modules.
2. Keep labels as structural anchors, not decorative pills.
3. Use thin frames, light rails, and soft depth to imply flow without heavy boxes.
4. Keep content abstract when the slide is about behavior rather than exact copy.

## Slide 8 Application

- Initial state: abstract question placeholders, LLM logo cluster, abstract answer placeholders.
- Transition state: Q/A pairs become compact cards grouped under `계획 / 설계 / 구현 / 검증`.
- Style: soft editorial deck language, translucent stage plate, precise repeated cards, minimal color.
