- `slide-style-kit.md`
  - AI에게 새 PPT/HTML 슬라이드 생성을 맡길 때 가장 먼저 제공할 스타일 지시문입니다.

- `assets/`
  - style kit 규칙에 포함된 공용 커버 배경과 로고 에셋입니다.
