# CDS / Components / Stepper

# [Usage] Stepper

Stepper는 상품 수량처럼 숫자 값을 직접 입력하거나 -, + 버튼으로 증감하기 위해 사용하는 입력 컴포넌트입니다.

## Context

Stepper는 사용자가 작은 범위의 수량을 빠르게 조정해야 할 때 사용해야 합니다. 자유로운 숫자 입력이 아니라 최소/최대 범위 안에서 증감하는 조작이므로, 수량 제한과 버튼 상태가 명확해야 합니다.

## Principle

- 수량 조작을 안정적으로 하기 위해 높이와 너비를 고정해야 합니다.
- 입력 가능한 숫자 범위를 명확히 제한해야 합니다.
- 감소/증가 버튼의 사용 가능 여부를 상태로 보여줘야 합니다.

## Usage Do / Don't

### Size

Do

- 고정된 높이와 너비를 사용해야 합니다.

Don't

- 콘텐츠 길이에 따라 Stepper 크기를 늘리거나 줄이지 않아야 합니다. (레이아웃 흔들림)

### Number

Do

- 최대 두 자리 수까지 입력할 수 있어야 합니다.

Don't

- 세 자리 수 입력을 허용하지 않아야 합니다. (규칙 초과)

### Control

Do

- -, + 버튼으로 값이 증감된다는 것을 명확히 보여줘야 합니다.

Don't

- 버튼 없이 숫자만 노출해 조작 가능성을 숨기지 않아야 합니다. (조작 가능성 인지 저하)

### Limit

Do

- 최소/최대 값에 도달하면 해당 방향 버튼을 비활성화해야 합니다.

Don't

- 더 이상 변경할 수 없는 방향의 버튼을 활성 상태로 두지 않아야 합니다. (오동작 기대)
