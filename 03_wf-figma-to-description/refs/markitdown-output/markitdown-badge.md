# [V2] Badge

배지는 탐색 항목 및 아이콘에 대한 알림, 카운트 또는 상태 정보를 표시합니다.

## Component

- 라벨이나 숫자를 포함할 수 있습니다.
- 내용은 "+"를 포함하여 4자로 제한합니다.

## Preview

### Light / Dark Fixed

| Light | Dark Fixed |
| --- | --- |
| 999+ / N | 999+ / N |

## Properties

### Dot Badge

| Properties | Description | Values |
| --- | --- | --- |
| Mode | Default를 선택하면 OS테마 환경에 따라 변경되고, Fixed를 선택하면 OS 테마 환경에 상관없이 다크테마로 보여집니다. | Default, Fixed |
| Size | Dot Badge의 사이즈를 정의합니다. | Small, Medium, Large |
| Outline | Dot Badge의 Outline 사용 여부를 정의합니다. | ON, OFF |

### Number Badge

| Properties | Description | Values |
| --- | --- | --- |
| Mode | Default를 선택하면 OS테마 환경에 따라 변경되고, Fixed를 선택하면 OS 테마 환경에 상관없이 다크테마로 보여집니다. | Default, Fixed |
| Type | Number Badge의 타입을 정의합니다. | Number, New |
| Label | Number Badge 라벨을 입력할 수 있습니다. | Typing |
