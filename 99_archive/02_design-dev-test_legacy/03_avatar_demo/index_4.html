[V2] Avatar Component Set

Figma node 62973:7556. Description의 Mode / Type / Size 축과 layout size를 기준으로 44개 조합을 정적 HTML/CSS variant board로 재구성했습니다.

index_4 · 44 variants
Mode(2) × Type(2) × Size(11), all combinations valid

Do

  • 개인 프로필이나 사용자 정체성 표현에는 Circle Avatar를 사용합니다.
  • 그룹, 솔로 아티스트, 공식 커뮤니티 대표 이미지에는 Squircle Avatar를 사용합니다.
  • 같은 리스트나 동일 계층 안에서는 동일한 Avatar size를 사용합니다.

Don't

  • 콘텐츠 목적이 다른 Avatar type을 임의로 혼용하지 않습니다.
  • 동일한 정보 계층 안에서 Avatar size를 섞어 레이아웃이 흔들리게 하지 않습니다.

[V2] Text Button Component Set

Figma node 52753:39618. Mode / Type / Size / Button Color / Status / Radius 축과 valid color 제약을 기준으로 576개 조합과 slot/content prop 샘플을 정적 HTML/CSS variant board로 추가했습니다.

Text Button · 576 variants

Do

  • Primary action에는 Filled 타입과 Primary Mint 컬러를 우선 사용합니다.
  • Secondary action에는 Outlined_gray 타입을 사용해 시각 위계를 낮춥니다.
  • 버튼 크기별 label spec, padding, height를 그대로 유지합니다.

Don't

  • Outlined_gray와 Ghost 타입에 Black button color를 임의로 추가하지 않습니다.
  • Loading 상태에서 중복 요청이 가능하도록 두지 않습니다.
  • 긴 label을 말줄임으로 핵심 action이 사라지게 만들지 않습니다.

[V2] Checkbox Component Set

Figma node 60365:40276. Description의 Mode / Type / Status 축과 token names를 기준으로 12개 조합을 정적 HTML/CSS variant board로 추가했습니다.

Checkbox · 12 variants

Do

  • 여러 항목을 0개부터 전체까지 선택할 수 있을 때 Check Box를 사용합니다.
  • 단일 항목의 동의, 승인, 확인에는 Check Box를 사용합니다.
  • 기본 Check Box는 Circle 타입을 사용합니다.

Don't

  • 여러 옵션 중 하나만 고르는 선택에 Check Box를 사용하지 않습니다.
  • Square와 Circle을 같은 선택 그룹 안에서 임의로 혼용하지 않습니다.
  • Default 상태에 회색 check icon을 노출하지 않습니다.

[V2] Radio Component Set

Figma node 59215:200965. Figma에서 확인한 Mode / Status 축과 24px geometry를 기준으로 6개 조합을 정적 HTML/CSS variant board로 추가했습니다.

Radio · 6 variants

Do

  • 여러 항목 중 하나만 선택해야 할 때 Radio를 사용합니다.
  • 같은 선택 그룹 안에서는 동일한 control 타입을 사용합니다.
  • 선택 전 상태는 채워지지 않은 빈 원으로 표시합니다.

Don't

  • 복수 선택 가능한 옵션에 Radio를 사용하지 않습니다.
  • Radio와 Check Box를 같은 선택 그룹 안에서 함께 사용하지 않습니다.
  • 선택 전 상태에 dot을 표시하지 않습니다.
